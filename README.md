Описание реализованного метода

1. Пользователь вводит количество чисел, которые будут находиться в списке
2. Пользователь вводит список из чисел
3. max = numbers[0] - берем первый элемент массива как начальное значение максимума
4. Проходим по всем элементам массива начиная со второго (i = 1) . Сравниваем каждый элемент с текущим максимумом
5. Если numbers[i] > max, то обновляем max = numbers[i]

Описание реализованного метода

1. Пользователь вводит количество чисел, которые будут находиться в списке
2. Пользователь вводит список из чисел
3. min = numbers[0] - берем первый элемент массива как начальное значение минимума
4. Проходим по всем элементам массива начиная со второго (i = 1) . Сравниваем каждый элемент с текущим минимумом
5. Если numbers[i] < min, то обновляем min = numbers[i]
